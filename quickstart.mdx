# Quickstart Guide

Follow these simple steps to start using Audentic in your project.

## Step 1: Sign Up

Create your Audentic account at [audentic.io](https://audentic.io)

## Step 2: Create Your First Agent

1. Navigate to the Agent Builder
2. Choose a template or start from scratch
3. Configure your agent's capabilities

## Step 3: Integration

Choose your preferred integration method and follow the setup instructions:

- Wix
- Shopify
- API
